# PRML 中文导读（prml-book）

本项目是 **Pattern Recognition and Machine Learning（PRML）** 的中文导读与学习笔记，使用 Quarto 构建为在线书籍。

## 在线阅读

- GitHub Pages: <https://bayesianovich.github.io/webbook/>

> 如果仓库从 `webbook` 重命名为 `prml-book`，Pages 地址通常会变为：
> <https://bayesianovich.github.io/prml-book/>

## 项目目标

- 按章节梳理 PRML 核心概念与公式。
- 用中文补充直觉解释、推导主线与学习路径。
- 提供可持续更新的在线阅读版本。

## 本地预览

需要先安装 Quarto：<https://quarto.org/docs/get-started/>

```bash
quarto preview
```

构建静态站点：

```bash
quarto render
```
